# MAIN INFERENCE FLOW
note over SL: Triggered via EventBridge
SL -> SNS: Publish Stock Batch (symbols)
SNS -> SQS: Fan-out Pattern (Filter Policy: PremiumTickers)

note over IW: Processed per Symbol
SQS -> IW: Deliver Symbol Message
IW -> API: Fetch current price + top 20 news articles
API -->> IW: Raw Data
IW -> IW: Calculate Decayed Sentiment Pulse
IW -> SM: Invoke Sentiment Model Endpoint
SM -->> IW: Sentiment score
IW -> SM: Invoke Ticker-Specific Inference Endpoint (e.g. price-model-AAPL-endpoint)
SM -->> IW: Price Model Probability
IW -> IW: Blend (1-w)Model + (w)Sentiment
IW -> DB: Save Results (conviction, modelScore, sentiment)
IW -> Cache: Invalidate Key (stock:SYMBOL)
note right of Cache: Forces frontend to fetch fresh AI signals
